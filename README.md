# IP-Addressing-Scheme
This python script is used to create an IP adressing scheme from a network ID 

----

for example...
182.168.0.0/24

-----------------------------------------------------------------------------

This will give you 

192.0.0.0/26
-----------------------------------------------------------------------------
192.0.0.64/26
-----------------------------------------------------------------------------
192.0.0.128/26
-----------------------------------------------------------------------------
192.0.0.192/26
-----------------------------------------------------------------------------

you can change it to adapt to your needs
